# AI Business Ideas — Full Analysis

> Дата: 2026-03-30
> Решение: **Aimatica** (aimatica.app) — AI Automation Platform

---

## Ключевые выводы

1. **AI-обёртки — commodity.** Любой может обернуть Claude/GPT API за выходные. Нет защиты от копирования.
2. **У большинства "очевидных" идей есть бесплатные альтернативы.** Shopify Magic (бесплатно, встроен), Canva AI (бесплатно), сам ChatGPT.
3. **Ни одна идея не набрала больше 6.2/10** при честной оценке по: спрос, конкуренция, распространение, доход, скорость MVP.
4. **Деньги там, где сложно скопировать:** уникальные данные, глубокая интеграция в workflow, network effects, доменная экспертиза.
5. **Лучший путь для соло-разработчика без PR/sales:** кастомные AI-автоматизации → замечаешь паттерны → упаковываешь в SaaS.
6. **Лучшие клиенты — бизнес, не блогеры.** Бизнес платит за измеримый ROI. Блогеры — плохие клиенты (мало зарабатывают, высокий churn, хотят бесплатно).

## Все идеи по категориям

### Developer tools (низкая маржа, высокая конкуренция)

- **AI Code Reviewer для CI/CD** — авто-ревью каждого PR через GitHub Action. Конкуренты: CodeRabbit ($15/user/мес), Sourcery, Qodo. Сложно выделиться.
- **AI Security Gate для CI/CD** — security-focused PR guard (OWASP, secrets, CVE, compliance). Лучше чем general review. Цель: fintech/medtech с PCI/HIPAA/SOC2 ($500-5000/мес за ручные аудиты).
- **AI миграция кодовых баз** — jQuery→React, Python 2→3
- **AI генератор API-документации** — код → OpenAPI + примеры
- **AI тест-генератор** — авто-генерация тестов при пуше
- **AI coding assistant** (Cursor clone) — рынок мёртв, нельзя конкурировать с компаниями с $400M+ инвестиций

### E-commerce

- **AI описания товаров для Shopify** — Shopify Magic БЕСПЛАТНО и встроен с 2023. Десятки конкурентов в App Store.
- **AI для e-commerce (общее)** — фото товаров → описания + SEO. 85% маржа. 1000 товаров × $0.50 = $500/клиент.

### Блоггинг / Контент

- **AI Thumbnail Generator** — Canva делает бесплатно. YouTubers заказывают на Fiverr за $5.
- **AI LinkedIn Ghost Writer** — Taplio, AuthoredUp, Kleo + 20 инструментов. Рынок забит.
- **AI Newsletter Writer** — Beehiiv добавляет AI. Авторы рассылок часто мало зарабатывают.
- **Video → Shorts/Reels AI** — Opus Clip ($29/мес), CapCut AI. Нужны GPU-серверы.
- **Full Content Repurposing Pipeline** — один вход → все платформы. 8 API поддерживать. Scope убивает соло-разработчиков.
- **AI текстовая генерация** (Jasper clone) — ChatGPT убил этот рынок.

### Бизнес-автоматизации

- **AI для бухгалтерии** — фото чека → проводка в 1С/QuickBooks. 90% маржа. Нужна доменная экспертиза.
- **AI для юристов** — анализ договора → список рисков. 90% маржа. Регуляторные риски.
- **AI для HR** — скрининг резюме + авто-письма. 85% маржа.
- **AI для клиник** — расшифровка приёма → карта пациента. 90% маржа. Регуляторная сложность.
- **AI для учителей** — генерация тестов, планов уроков, проверка работ.
- **AI для недвижимости** — описания объектов, анализ рынка, чат-бот для клиентов.
- **AI переводчик тех. документации** — контекстно-зависимый перевод (лучше DeepL).
- **AI генератор лендингов** — описание → рабочий сайт за 5 минут.
- **AI видео-редактор** — авто-субтитры + монтаж (Electron + Whisper).
- **AI дизайнер соцсетей** — генерация баннеров/постов для SMM.

### Новые идеи (последний раунд)

- **AI Review Manager для локального бизнеса** — авто-ответы на Google/Yelp/TripAdvisor отзывы. $20-50/мес. Конкуренты: BirdEye, Podium ($300+/мес). Нужен продажник.
- **AI Data Analyst для малого бизнеса** — подключил Google Sheets → задаёшь вопросы → графики + рекомендации. $30-100/мес. Огромный TAM.
- **AI Meeting → Action** — не транскрипция, а создание задач в Jira, follow-up email, обновление wiki. $20-50/user/мес.
- **AI бэк-офис фрилансера** — авто-proposals для Upwork, договоры, счета, переписка. $15-30/мес. 70M+ фрилансеров в США.

## Рекомендованный путь: Automation → SaaS

### Phase 1: Кастомные AI-автоматизации ($0 стартовых)
Найди бизнес с повторяющейся ручной работой. Построй кастомную автоматизацию.
- $500-2000 за настройку + $100-300/мес поддержка
- 10 клиентов = $1,000-3,000/мес recurring

### Phase 2: Заметь паттерны
После 5-10 клиентов 3 из 5 просят одно и то же.

### Phase 3: Упакуй в SaaS
Самый частый запрос → self-serve продукт → запуск.

### Примеры автоматизаций
- Онлайн-школа: AI обрабатывает заявки, квалифицирует лиды, записывает на пробный урок
- Продавец на маркетплейсе: AI мониторит цены конкурентов → автокорректировка
- Рекрутинговое агентство: AI парсит резюме → матчит с вакансиями → пишет outreach
- Интернет-магазин: AI-чатбот подключённый к реальному каталогу товаров
